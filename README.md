# ATK-mu-Fotocopy-store-app-android-
ATK-mu (Fotocopy store app android)
This app is Fotocopy store online in android  base. This application like Shopee, Tokopedia, Bukalapak and so on.
Components :
1.	Android Studio
2.	Web Server
3.	Google Developer Account
4.	NoSQL Database (JSON)
